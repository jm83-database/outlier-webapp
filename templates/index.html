사용 가이드

🚀 시작하기

1단계: 샘플명, 생산일자, 패스 수를 입력하세요
2단계: 데이터를 직접 입력하거나 Excel/CSV 파일을 업로드하세요
3단계: 임계값을 조정하고 "이상치 검출 및 계산" 버튼을 클릭하세요

📊 데이터 입력 방법

✏️ 직접 입력
  • 표의 셀에 직접 숫자를 입력
  • "행 추가" 버튼으로 데이터 추가
  • "컬럼 추가" 버튼으로 새로운 측정 항목 추가
📁 파일 업로드
  • Excel (.xlsx, .xls) 또는 CSV 파일 지원
  • size, nm, PI 컬럼 자동 매핑
  • 기타 숫자 컬럼도 자동 인식

🎯 이상치 검출 방법

Z-Score

평균에서 표준편차의 몇 배 떨어져 있는지 측정

권장값: 2.0~3.0

높을수록: 더 관대한 기준

IQR

사분위수 범위를 벗어나는 데이터 검출

권장값: 1.5~2.0

높을수록: 더 관대한 기준

MAD

중앙값 절대편차 기반 검출

권장값: 3.0~4.0

높을수록: 더 관대한 기준

💾 데이터 관리

📂 저장 및 불러오기
  • "데이터셋 저장"으로 현재 작업 저장
  • "데이터셋 불러오기"로 이전 작업 복원
  • 여러 실험 데이터를 개별 관리
📈 비교 분석
  • "데이터셋 비교"로 여러 실험 비교
  • 시각적 그래프로 차이점 확인
  • 통계 요약 테이블 제공

📋 결과 활용

📊 시각화

인터랙티브 산점도로 데이터 분포 확인, 마우스 오버로 세부 정보 표시

📁 결과 다운로드

"결과 다운로드" 버튼으로 CSV 파일 저장, 원본 데이터와 처리 결과 모두 포함

💡 팁과 주의사항

✅ 좋은 사용법
  • 데이터 입력 전 샘플명과 날짜 먼저 설정
  • 임계값을 여러 번 조정해보며 최적값 찾기
  • 중요한 데이터는 반드시 저장하기
  • 다양한 방법으로 이상치 검출 비교
⚠️ 주의사항
  • 빈 셀이나 문자는 자동으로 제외됨
  • 최소 2개 이상의 유효한 데이터 필요
  • 새로고침 시 저장되지 않은 데이터 손실
  • 파일 업로드는 16MB 이하만 가능

🔄 예제 워크플로우

  1. 실험 정보 입력: "Sample_A", "2024-01-15", 패스 1
  2. 데이터 입력: 직접 입력 또는 Excel 파일 업로드
  3. 임계값 조정: Z-Score 3.0, IQR 1.5, MAD 3.5
  4. 계산 실행: "이상치 검출 및 계산" 클릭
  5. 결과 확인: 시각화와 통계 테이블 검토
  6. 데이터 저장: "Sample_A_Pass1"로 저장
  7. 결과 다운로드: CSV 파일로 보고서 작성용 다운로드

추가 문의사항이 있으시면 시스템 관리자에게 연락하세요.
