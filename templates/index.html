사용 가이드

🚀 시작하기

1단계: 샘플명, 생산일자, 패스 수(회차, iteration)를 입력하세요
2단계: 데이터를 직접 입력하거나 Excel/CSV 파일을 업로드하세요
3단계: 임계값을 조정하고 "이상치 검출 및 계산" 버튼을 클릭하세요

📊 데이터 입력 방법

✏️ 직접 입력
  • 표의 셀에 직접 숫자를 입력
  • "행 추가" 버튼으로 데이터 추가
  • "컬럼 추가" 버튼으로 새로운 측정 항목 추가
📁 파일 업로드
  • Excel (.xlsx, .xls) 또는 CSV 파일 지원
  • size, µm, PI 컬럼 자동 매핑
  • 기타 숫자 컬럼도 자동 인식

🎯 이상치 검출 방법

Z-Score

평균에서 표준편차의 몇 배 떨어져 있는지 측정

권장값: 2.0~3.0

높을수록: 더 관대한 기준

IQR

사분위수 범위를 벗어나는 데이터 검출

권장값: 1.5~2.0

높을수록: 더 관대한 기준

MAD

중앙값 절대편차 기반 검출

권장값: 3.0~4.0

높을수록: 더 관대한 기준

💾 데이터 관리

📂 저장 및 불러오기
  • "데이터셋 저장"으로 현재 작업 저장
  • "데이터셋 불러오기"로 이전 작업 복원
  • 여러 실험 데이터를 개별 관리
📈 비교 분석
  • "데이터셋 비교"로 여러 실험 비교
  • 시각적 그래프로 차이점 확인
  • 통계 요약 테이블 제공

📈 패스별 데이터 관리

📝 평균값 입력
  • 수동 입력: 패스 번호, Size(µm) 평균, PI 평균 직접 입력
  • 계산 결과 활용: 이상치 검출 결과에서 자동 추가
  • 점도 데이터: 선택적으로 점도값 (cP) 함께 저장
📊 트렌드 분석
  • 패스별 트렌드: Size/PI 값의 패스별 변화 추이
  • 상관관계 분석: Size-PI 간 상관계수 계산
  • 통계 요약: 평균, 표준편차, 변동계수 제공

📋 결과 활용

📊 시각화

인터랙티브 산점도로 데이터 분포 확인, 마우스 오버로 세부 정보 표시

📁 결과 다운로드

"결과 다운로드" 버튼으로 CSV 파일 저장, 원본 데이터와 처리 결과 모두 포함

💡 팁과 주의사항

✅ 좋은 사용법
  • 데이터 입력 전 샘플명과 날짜 먼저 설정
  • 임계값을 여러 번 조정해보며 최적값 찾기
  • 중요한 데이터는 반드시 저장하기
  • 다양한 방법으로 이상치 검출 비교
⚠️ 주의사항
  • 빈 셀이나 문자는 자동으로 제외됨
  • 최소 2개 이상의 유효한 데이터 필요
  • 새로고침 시 저장되지 않은 데이터 손실
  • 파일 업로드는 16MB 이하만 가능

🧪 점도 데이터 관리

📊 점도 데이터 입력
  • 패스별 입력: 패스별 평균값 폼에서 점도 (cP) 입력
  • 선택사항: Size/PI 평균값과 함께 점도값 저장
  • 테이블 확인: 입력된 점도값은 패스별 데이터 테이블에 표시
📈 점도 상관관계 분석
  • 분석 조건: 점도 데이터 2개 이상 필요
  • 시각화: Size(μm) vs 점도 산점도 차트
  • 기준값 비교: UHV, HV, LV 기준값과 생산 데이터 비교
  • 상관계수: Size-점도 간 상관관계 수치 제공
💡 점도 데이터 활용 가이드
  1. 1단계: 이상치 검출로 패스별 Size(µm) 평균값 계산
  2. 2단계: 패스별 평균값 폼에서 Size/PI/점도값 입력
  3. 3단계: "점도 상관관계 분석" 버튼으로 시각화 확인
  4. 4단계: 기준값 대비 생산 데이터 위치 분석

🔄 예제 워크플로우

  1. 실험 정보 입력: "Sample_A", "2024-01-15", 패스 1
  2. 데이터 입력: 직접 입력 또는 Excel 파일 업로드
  3. 임계값 조정: Z-Score 3.0, IQR 1.5, MAD 3.5
  4. 계산 실행: "이상치 검출 및 계산" 클릭
  5. 결과 확인: 시각화와 통계 테이블 검토
  6. 패스별 데이터 저장: Size/PI 평균값과 점도값 입력
  7. 점도 분석: "점도 상관관계 분석"으로 시각화 확인
  8. 데이터 저장: "Sample_A_Pass1"로 저장
  9. 결과 다운로드: CSV 파일로 보고서 작성용 다운로드
