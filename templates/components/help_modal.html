사용 가이드

🚀 시작하기

1단계: 샘플명, 생산일자, 패스 수(회차, iteration)를 입력하세요
2단계: 데이터를 직접 입력하거나 Excel/CSV 파일을 업로드하세요
3단계: 임계값을 조정하고 "이상치 검출 및 계산" 버튼을 클릭하세요
4단계: 결과 확인 후 데이터셋 저장 또는 다운로드하세요

📊 데이터 입력 방법

✏️ 직접 입력
  • 표의 셀에 직접 숫자를 입력
  • "행 추가" 버튼으로 데이터 추가
  • "컬럼 추가" 버튼으로 새로운 측정 항목 추가
📁 파일 업로드
  • Excel (.xlsx, .xls) 또는 CSV 파일 지원
  • size, nm, PI 컬럼 자동 매핑
  • 기타 숫자 컬럼도 자동 인식

🎯 이상치 검출 방법

Z-Score

평균에서 표준편차의 몇 배 떨어져 있는지 측정

권장값: 2.0~3.0

높을수록: 더 관대한 기준

IQR

사분위수 범위를 벗어나는 데이터 검출

권장값: 1.5~2.0

높을수록: 더 관대한 기준

MAD

중앙값 절대편차 기반 검출

권장값: 3.0~4.0

높을수록: 더 관대한 기준

💾 데이터셋 관리

📂 저장 및 불러오기
  • 데이터셋 저장: 이름 입력 후 현재 작업 저장
  • 데이터셋 불러오기: 드롭다운에서 선택 후 이전 작업 복원
  • 데이터셋 삭제: 확인 후 불필요한 데이터 안전하게 제거
  • 저장된 개수 표시: 현재 저장된 데이터셋 개수 실시간 확인
  • 자동 지속성: 저장된 데이터는 브라우저 세션 간 유지
  • 여러 실험 데이터를 개별 관리
📊 데이터셋 비교
  • 다중 선택: 체크박스로 여러 데이터셋 선택
  • 시각화 비교: 선택한 데이터셋들을 하나의 차트로 비교
  • 통계 요약: 각 데이터셋의 평균, 표준편차 등 비교
  • 최소 조건: 비교하려면 최소 2개의 데이터셋 필요
  • 결과 위치: 데이터셋 관리 섹션 바로 아래에 표시
💡 데이터셋 관리 워크플로우
  1. 데이터 입력: 샘플 정보 및 측정 데이터 입력
  2. 이상치 검출: 임계값 설정 후 계산 실행
  3. 결과 저장: 분석 결과를 의미 있는 이름으로 저장
  4. 데이터 비교: 여러 실험 결과 간 비교 분석 (결과는 데이터셋 관리 섹션 아래 표시)
  5. 결과 활용: 패스별 관리 또는 보고서 작성

📈 데이터 다운로드

🔄 기본 다운로드
  • 테이블 다운로드: 클릭 한 번으로 즉시 CSV 다운로드
  • 스마트 업로드: 어떤 형식이든 자동 파싱
  • 완벽한 호환: 다운로드 → 편집 → 업로드 무제한 반복
🔍 분석 결과 다운로드
  • 이상치 결과 다운로드: 이상치 제거 후 분석 결과만
  • 🆕 데이터+이상치 결과 다운로드: 원본 데이터와 각 방법별 이상치 판정 결과 통합
🔄 간단한 편집 워크플로우
  1. 다운로드: "테이블 다운로드" 버튼 클릭
  2. 편집: Excel/메모장에서 자유롭게 수정
  3. 업로드: 수정된 파일 업로드
  4. 완료: 스마트 파싱으로 자동 로드

✨ 복잡한 옵션 선택 없이 클릭 한 번으로 바로 다운로드!

🧪 실험군/대조군 데이터 관리

📝 실험군 데이터 입력
  • 샘플명: 실험 샘플의 고유 식별자
  • 생산일자: 실험 데이터의 생산 날짜 (시각화 그룹핑)
  • 이상치 제거법: Manual, Z-Score, IQR, MAD 중 선택
  • Size/PI 평균: 입자 크기 및 다분산지수 평균값
  • 사용자 정의 필드: 점도, 온도, 압력 등 측정 데이터
  • 계산 결과 활용: 이상치 검출 결과에서 자동 추가
📊 대조군 데이터 입력 (레퍼런스)
  • 샘플명: 대조군 샘플의 고유 식별자
  • Size/PI 평균: 기준 입자 크기 및 다분산지수
  • 사용자 정의 필드: 기준 측정 데이터
  • 생산일자 불필요: 레퍼런스이므로 날짜 입력 없음
  • 시각화 구분: 'Reference Values'로 표시
💡 이상치 제거법 정확도 향상
  • 명시적 선택: 사용자가 직접 이상치 제거법을 선택하여 정보 정확도 향상
  • 자동 감지 보조: Manual 선택 시에만 계산 결과와 매칭하여 자동 감지
  • 중복 방지: Z-Score와 IQR 결과가 동일해도 사용자 선택 방법 우선 적용
  • 임계값 매칭: 선택한 방법의 계산 결과와 일치하는 경우 임계값 정보 자동 추가

📊 샘플별 트렌드 분석 및 시각화

📈 트렌드 분석 기능
  • 샘플별 트렌드: Size/PI 값의 샘플별 변화 추이
  • 생산일자별 정렬: 실험군 데이터를 생산일자 순으로 정렬
  • 상관관계 분석: Size-PI 간 상관계수 계산
  • 통계 요약: 평균, 표준편차, 변동계수 제공
  • 실험군/대조군 구분: 실험군(파란색), 대조군(Reference Values, 초록색)
🎨 시각화 개선사항
  • 날짜별 색상 구분: 실험군 데이터를 생산일자별로 색상 구분
  • 마커 구분: 실험군(원형), 대조군(다이아몬드)
  • 범례 최적화: "Aa" 텍스트 제거로 깔끔한 범례
  • 호버 정보: 샘플명, 생산일자, 측정값 포함
  • Reference Values: 모든 차트에서 대조군 일관된 표시
💡 사용자 정의 필드 상관관계 분석 가이드
  1. 실험군 입력: 샘플명, 생산일자, 이상치 제거법 등 포함하여 입력
  2. 대조군 입력: 레퍼런스 데이터로 생산일자 없이 입력
  3. 상관관계 분석: 사용자 정의 필드 상관관계 분석 버튼 클릭
  4. 시각화 확인: 날짜별 색상 구분된 실험군과 다이아몬드 모양 대조군 확인
  5. 통계 해석: 실험군/대조군별 통계 정보 및 상관계수 확인

🔒 시크릿 모드로 접속하세요:

  • Chrome/Edge: Ctrl + Shift + N
  • Firefox: Ctrl + Shift + P
  • Safari: Cmd + Shift + N

2단계: 새로고침 버튼 사용
  • 화면 상단 우측의 "🔄 새로고침" 버튼을 클릭하세요
  • 페이지가 완전히 새로 로드되면서 모든 캐시가 정리됩니다
  • 일반적인 기능 오류나 화면 문제가 해결됩니다
3단계: 다크모드 토글 버튼 사용
  • 화면 상단 좌측의 "🌙/☀️ 다크모드" 토글 버튼을 클릭하세요
  • 다크모드와 라이트모드를 자유롭게 전환할 수 있습니다
  • 설정은 자동으로 저장되어 다음 방문 시에도 유지됩니다
  • 어두운 환경에서 눈의 피로를 줄여주는 다크모드를 활용하세요

브라우저는 웹사이트 속도를 향상시키기 위해 파일들을 임시 저장(캐시)합니다. 하지만 웹사이트가 업데이트되면 이전 버전이 계속 사용되어 문제가 발생할 수 있습니다.

💡 팁과 주의사항

✅ 좋은 사용법
  • 데이터 입력 전 샘플명과 날짜 먼저 설정
  • 임계값을 여러 번 조정해보며 최적값 찾기
  • 중요한 데이터는 반드시 저장하기
  • 다양한 방법으로 이상치 검출 비교
  • 문제 발생 시 먼저 새로고침 버튼 클릭
⚠️ 주의사항
  • 빈 셀이나 문자는 자동으로 제외됨
  • 최소 2개 이상의 유효한 데이터 필요
  • 새로고침 시 저장되지 않은 데이터 손실
  • 파일 업로드는 16MB 이하만 가능
  • 시크릿 모드에서는 저장된 데이터셋 접근 불가

🧪 사용자 정의 필드 관리

📊 사용자 정의 데이터 입력
  • 필드명 설정: 기본 "사용자 입력 필드(레퍼런스)"를 원하는 이름으로 변경
  • 데이터 종류: 점도, 온도, 압력, 농도 등 어떤 데이터든 가능
  • 패스별 입력: 패스별 평균값 폼에서 레퍼런스값 입력
  • 테이블 확인: 입력된 값은 패스별 데이터 테이블에 표시
📈 상관관계 분석
  • 분석 조건: 사용자 정의 데이터 2개 이상 필요
  • 시각화: Size(nm) vs 사용자 정의 필드 산점도 차트
  • 기준값 비교: 점도 데이터인 경우 UHV, HV, LV 기준값과 비교
  • 상관계수: Size-사용자 정의 필드 간 상관관계 수치 제공
💡 사용자 정의 필드 활용 가이드
  1. 1단계: 입력 필드명을 원하는 이름으로 변경 (예: "온도(℃)", "압력(bar)")
  2. 2단계: 이상치 검출로 패스별 Size(nm) 평균값 계산
  3. 3단계: 패스별 평균값 폼에서 Size/PI/사용자 정의 값 입력
  4. 4단계: "상관관계 분석" 버튼으로 시각화 확인
  5. 5단계: 필요시 기준값 대비 생산 데이터 위치 분석

🔄 예제 워크플로우

  1. 실험 정보 입력: "Sample_A", "2024-01-15", 패스 1
  2. 데이터 입력: 직접 입력 또는 Excel 파일 업로드
  3. 임계값 조정: Z-Score 3.0, IQR 1.5, MAD 3.5
  4. 계산 실행: "이상치 검출 및 계산" 클릭
  5. 결과 확인: 시각화와 통계 테이블 검토
  6. 데이터셋 저장: 결과를 "Sample_A_Pass1"로 저장
  7. 패스별 관리: 실험군/대조군 데이터 입력 및 상관관계 분석
  8. 다중 실험 비교: 여러 패스나 샘플 간 비교 분석
  9. 결과 다운로드:
    • "이상치 결과 다운로드": 정제된 데이터만
    • "데이터+이상치 결과 다운로드": 원본 데이터와 이상치 판정 결과
